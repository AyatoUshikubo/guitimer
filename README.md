# Quickstart for Mac
1.Install Xcode.  
2.Install Homebrew.  
3.Install ROOT.  
```$ brew install root```  
4.Move to the 'guitimer' directory.  
5.Run the 'developtimer.C' file.  
```$ root developtimer.C```
