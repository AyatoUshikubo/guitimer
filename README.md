## GUI Timer
### Start
タイマーがスタートし、Unlockボタンに切り替わる。
### Unlock
Resetボタンが使用可能になり、Lockボタンに切り替わる。
### Lock
Resetボタンが使用不可になり、Unlockボタンに切り替わる。
### Reset
タイマーがスタート前に戻る。
